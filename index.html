This visualizer was created by Jeffrey Kim, a student studying Electrical Engineering and Computer Science at the University of California, Berkeley.
It randomly creates 3 datasets using the selected parameters and trains each model on every dataset. It then generates the corresponding heatmaps using the posterior probabilities of each model.
